Entendendo sobre HTML

Sobre a tag h1 até h6:

representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.

Sobre a tag p:

A tag "p" é utilizada para demarcar um parágrafo.

Sobre a tag Mark:

representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Ex abaixo:

HTML

Sobre a tag Small:

é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto.

Sobre a tag I:

pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo "itálico".

Sobre a tag U:

Ela é usada para sublinhar o "texto" dentro de um elemento específico.

Sobre a tag strong:

dá ao texto uma forte importância, e é tipicamente mostrado em "negrito".

Sobre a tag OL:

representa uma lista de itens ordenados. Ex abaixo:

    Listagem 1
    Listagem 2

Sobre a tag UL:

Representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. Ex abaixo:

    Teste

Sobre a tag LI:

É usado para representar um item que faz parte de uma lista. Ex abaixo:

  • Item 1
  • Item 2

    Representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Ex abaixo:

    HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.     Sobre a tag SUB:

    É utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo. Ex abaixo:

    "HTML"

    Sobre a tag SUP:

    é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo. Ex abaixo:

    "HTML"
